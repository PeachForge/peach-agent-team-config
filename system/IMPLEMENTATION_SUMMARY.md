## 核心功能

### 升级流程
1. ✅ 检查新版本（Git tag/配置文件版本）
2. ✅ 备份当前版本
3. ✅ 下载新版本
4. ✅ 迁移配置
5. ✅ 验证升级
6. ✅ 清理旧备份（保留最近 3 个）

### 回滚流程
1. ✅ 检测升级失败
2. ✅ 恢复最近备份
3. ✅ 记录失败原因
4. ✅ 发送告警

```bash

# 查看状态
```
